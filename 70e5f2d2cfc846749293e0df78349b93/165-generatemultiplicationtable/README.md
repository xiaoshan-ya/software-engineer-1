Generate multiplication table

>1.生成一个从0到n的乘法表
>n是一个正整数

>2.程序从控制台获取输入n
>
>输出的顺序应当按照乘法表顺序由小到大
>其中乘号用大写的X表示
>输出时注意空格与换行


项目说明：

>src目录下为题目代码源文件，需要完成的内容为generate_multiplication_table.py里的方法。
>src目录下的__init__.py文件不需要改动。
>
>test目录下为测试用例文件，不可进行修改。


示例：

	输入：
	  n=2

	输出：
	  0 X 0 = 0
	  0 X 1 = 0
	  0 X 2 = 0
	  1 X 0 = 0
	  1 X 1 = 1
	  1 X 2 = 2
	  2 X 0 = 0
	  2 X 1 = 2
	  2 X 2 = 4
